# 微分積分方程式 数値シミュレータ (Integro-Differential Equation Solver)

Rustと`egui`（`eframe`）を用いて構築された、純粋数学的な微分積分方程式の数値解法シミュレータです。方程式を解くだけでなく、微分項・積分項のそれぞれの独立した解（数値解）をリアルタイムに分離・評価することができます。

## 📌 対象とする方程式

本アプリケーションは、以下の一般的なボルテラ型微分積分方程式をモデル化しています。

$$f'(x) = k \cdot g(x) - \lambda \int_{0}^{x} f(t) \, dt \quad \text{ただし、初期条件 } f(0) = 0$$

- **$k$**: 外部入力の係数（システムへのゲイン）
- **$\lambda$**: 積分項の重み係数（過去の履歴が現在の変化量に与える影響度）
- **$g(x)$**: 定数関数（本アプリでは $g(x) = 1.0$ 固定）

> 💡 **数学的特例**: $k = 1.0, \lambda = 1.0$ のとき、この方程式の数学的厳密解（解析解）は $f(x) = \sin(x)$ に一致します。このとき、積分の解は $1 - \cos(x)$、微分の解は $\cos(x)$ となります。

## ✨ 主な機能

- **個別解の分離・抽出**: 関数値 $f(x)$ だけでなく、内部で計算されている「積分の解」「微分の解」を個別にリアルタイム出力。
- **インタラクティブなパラメータ調整**: スライダーやドラッグ値を用いて、ターゲットとする $x$ の値や各種係数を直感的に変更可能。
- **高速な数値解析**: 内部ロジックにはオイラー法と台形公式（分割数 $N=1000$）を組み合わせ、Rustのパワーを活かした高速かつ安全な反復計算を実現。
- **日本語フォント対応**: `egui`独自の文字化け（豆腐文字）対策として、起動時にシステムフォントを自動的に読み込む仕組みを導入。

## 🛠️ 動作環境と要件

- **言語**: Rust (Edition 2021)
- **主要依存クレート**: `eframe` / `egui` (v0.27以降)
- **対応OS**: Windows (OS標準の `msgothic.ttc` を参照)
  - ※MacやLinux環境で実行する場合は、`src/main.rs` 内の `font_path` をそれぞれの環境のフォントパス（例: `HiraginoSans-W3.ttc` など）に書き換えてください。
